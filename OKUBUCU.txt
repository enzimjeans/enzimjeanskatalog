🚀 ILKEZ KULLANIM (Yeni Bilgisayarda)
═══════════════════════════════════════════════════════

1. Bu klasoru istediginiz yere kopyalayin:
   - Masaustu
   - Belgeler
   - USB bellek
   - Herhangi bir yer!

2. "ilk_kurulum.bat" dosyasina cift tiklayin
   - Git kurulumu kontrol edilir
   - Excel2JSON.exe kontrol edilir (Python gerekmez!)
   - GitHub baglantisi OTOMATIK kurulur!
   - mayakozmetikbijuteri-prog repository'e baglanir
   - 5-10 dakika surer

   ONEMLI: GitHub Token gereklidir!
   - GitHub Settings -> Tokens
   - Generate new token (classic)
   - "repo" yetkisi secin
   - Token'i kopyalayin ve kaydedin

   NOT: GitHub bilgileri otomatik ayarlanir,
        hicbir sey girmenize gerek yok!

3. "config.js" dosyasini duzenleyin:
   - Not defteri ile acin
   - WhatsApp numaranizi girin: +905XXXXXXXXX
   - Kaydedin

4. Ilk push yaparken:
   - guncelle.bat'a basin
   - Sifre istediginde TOKEN'i girin (sifre degil!)

5. TAMAM! Artik kullanabilirsiniz!

═══════════════════════════════════════════════════════
❓ SIK SORULAN SORULAR
═══════════════════════════════════════════════════════

S: Python yok hatasi aliyorum?
C: Excel2JSON.exe dosyasi varsa sorun yok. Emin olmak icin
   klasorde "Excel2JSON.exe" dosyasini arayın.

S: Baska bilgisayarda nasil kullanırım?
C: Tum klasoru USB ile kopyalayin, "ilk_kurulum.bat" calistirin.

S: Excel acik hatasi aliyorum?
C: Excel'i mutlaka kapatin, sonra guncelle.bat'a basin.

S: Site guncellenmiyor?
C: 2-3 dakika bekleyin, tarayiciyi Ctrl+F5 ile yenileyin.

S: Gorseller gozukmuyor?
C: - Dosya adlarini kontrol edin
   - Buyuk/kucuk harf onemli
   - Gorseller "gorseller/" klasorunde olmali

S: Sepet sistemi nasil calisıyor?
C: Musteri urun secer, sepete ekler, WhatsApp'tan gonderir.

S: WebP gorsel yukleyebilir miyim?
C: Evet! jpg, png, webp, gif, svg hepsi desteklenir.
